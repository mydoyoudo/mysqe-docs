# 📚 行业标准速查

## ISO 9001:2015 核心要点

### 7 大原则
1. 以客户为关注焦点
2. 领导作用
3. 全员参与
4. 过程方法
5. 改进
6. 循证决策
7. 关系管理

### PDCA 循环
```
Plan (计划) → Do (执行) → Check (检查) → Act (处理)
```

### 10 条款结构
| 条款 | 名称 |
|------|------|
| 4 | 组织环境 |
| 5 | 领导作用 |
| 6 | 策划 |
| 7 | 支持 |
| 8 | 运行 |
| 9 | 绩效评价 |
| 10 | 改进 |

## IATF 16949:2016 汽车质量管理体系

### 7 大工具
1. APQP - 产品质量规划
2. PPAP - 生产件批准程序
3. FMEA - 失效模式与影响分析
4. SPC - 统计过程控制
5. MSA - 测量系统分析
6. CONTROL PLAN - 控制计划
7. PMBOK (不是，纠正措施通常是 APQP 的一部分)

### 汽车行业特殊要求
- 顾客特殊要求 (CSR)
- 过程符号 (乌龟图)
- 产品安全
- 法规符合性
- 供应链管理

## 常用质量术语

| 术语 | 英文 | 定义 |
|------|------|------|
| 合格率 | Yield | 合格品数 / 总数 × 100% |
| 一次合格率 | FPY | 第一次就合格的比例 |
| 返工率 | Rework Rate | 返工数 / 总数 × 100% |
| 客诉率 | Complaint Rate | 客诉次数 / 出货次数 |
| PPM | Parts Per Million | 每百万件不良数 |
| CPK | Process Capability Index | 过程能力指数 |
| GR&R | Gauge R&R | 量具重复性和再现性 |
| OQC | Outgoing Quality Control | 出货检验 |
| IQC | Incoming Quality Control | 来料检验 |
| FQC | Final Quality Control | 最终检验 |
