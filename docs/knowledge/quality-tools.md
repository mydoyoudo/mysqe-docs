# 📖 质量知识库

## 五大核心工具

### 1. APQP (产品质量规划)

| 阶段 | 输出物 | 关键活动 |
|------|--------|----------|
| 1. 计划和定义 | 项目章程、质量目标 | 市场需求分析 |
| 2. 产品设计开发 | DFMEA、特殊特性 | 设计评审 |
| 3. 过程设计开发 | PFMEA、流程图 | 工艺参数确定 |
| 4. 产品和过程确认 | 试生产、MSA | 量产准备 |
| 5. 反馈、评定和纠正措施 | 量产、持续改进 | 质量优化 |

### 2. PPAP (生产件批准程序)

**提交等级**：
- Level 1: 仅提交保证书
- Level 2: 提交保证书 + 样件
- Level 3: 提交保证书 + 样件 + 关键材料
- Level 4: 提交保证书 + 样件 + 全部材料
- Level 5: 提交保证书 + 样件 + 全部材料 + 现场审核

**必须提交项** (18项)：
1. 设计记录
2. 工程变更文件
3. 客户工程批准
4. DFMEA/PFMEA
5. 过程流程图
6. PFMEA
7. 过程指导书
8. 测量系统分析 (MSA)
9. 初始过程研究 (PPK)
10. 合格实验室文件
11. 外观批准报告
12. 生产件样品
13. 标准样品
14. 检查治具
15. 客户特殊要求
16. 零件提交保证书 (PSW)
17. 散装材料要求检查表
18. 鞋带检查表 (如有)

### 3. FMEA (失效模式与影响分析)

#### 结构树
```
系统 → 子系统 → 部件 → 零件
```

#### 严重度 (S) 评分标准

| 评分 | 严重度 | 定义 |
|------|--------|------|
| 10 | 无警告的严重 | 安全项目潜在失效 |
| 9 | 有警告的严重 | 法规项目潜在失效 |
| 8 | 很高 | 功能丧失 |
| 7 | 高 | 功能降级 |
| 6 | 中等 | 舒适性问题 |
| 5 | 低 | 轻微功能丧失 |
| 4 | 很低 | 轻微功能降级 |
| 3 | 轻微 | 轻微问题 |
| 2 | 极轻微 | 极轻微问题 |
| 1 | 无 | 无可识别影响 |

#### 发生度 (O) 评分标准

| 评分 | 发生度 | 失效概率 |
|------|--------|----------|
| 10 | 极高 | ≥1/10 |
| 9 | 很高 | 1/20 |
| 8 | 高 | 1/50 |
| 7 | 中高 | 1/100 |
| 6 | 中等 | 1/200 |
| 5 | 中低 | 1/500 |
| 4 | 低 | 1/1000 |
| 3 | 很低 | 1/10000 |
| 2 | 极低 | 1/100000 |
| 1 | 无 | 0 |

#### 探测度 (D) 评分标准

| 评分 | 探测度 | 探测可能性 |
|------|--------|------------|
| 10 | 无 | 无法探测 |
| 9 | 极低 | 很低概率发现 |
| 8 | 低 | 发现概率低 |
| 7 | 中低 | 中等发现概率 |
| 6 | 中等 | 中等发现概率 |
| 5 | 中高 | 较高发现概率 |
| 4 | 高 | 很容易发现 |
| 3 | 很高 | 几乎肯定发现 |
| 2 | 极高 | 确定发现 |
| 1 | 绝对确定 | 100% 发现 |

#### RPN 优先级
- **高风险**: RPN ≥ 100 → 立即采取行动
- **中风险**: 80 ≤ RPN < 99 → 计划采取行动
- **低风险**: RPN < 80 → 监控

### 4. SPC (统计过程控制)

#### 控制图类型

| 数据类型 | 适用范围 | 控制图 |
|----------|----------|--------|
| 计量型 | 连续数据 | X̄-R, X̄-S, I-MR |
| 计数型 | 不合格数 | np, p, c, u |

#### 控制图判异准则
1. 点超出 3σ 限
2. 连续 7 点在中心线同侧
3. 连续 6 点递增/递减
4. 连续 14 点交替上下
5. 连续 3 点中有 2 点在 A 区
6. 连续 5 点中有 4 点在 B 区外
7. 连续 15 点在 C 区

### 5. MSA (测量系统分析)

#### GR&R 判定标准

| GR&R% | 判定 |
|-------|------|
| < 10% | 优秀 |
| 10-30% | 可接受 |
| > 30% | 不可接受，需改进 |

#### 研究类型
- **极差法**: 2-3 操作员，快速评估
- **均值-极差法**: 标准方法，5-10 样本
- **方差分析法 (ANOVA)**: 最精确，分离人和仪器变异
